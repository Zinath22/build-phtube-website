1.Discuss the scope of var, let, and const

Var:

Var variables can be updated and re-declared within its scope.Its keyword is global

Let:

The scope of a let variable is only block scoped.It can not be accessible outside the particular block.

Const:

Const variables can neither be updated nor re-declared.when user declare a const variable,they need to initialized it.otherwise it returns error

2.Tell us the use cases of null and undefined

Null:

It is one of primitive values of JS.It is the intentional absence of the value

undefined:

It means the value does not exist in a compiler.It is a global object
